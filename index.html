TriMet Rail Operations Optimization Technology (ROOT)

Rail Operations Optimization Technology, or ROOT, will use the Internet of Things and Big Data Analytics to optimize rail operations at TriMet. This dashboard provides a snapshot of ROOT’s progress. These key indicators help us track progress toward project deliverables and the budget.

More information about ROOT can be found by clicking the About ROOT button below. Information about the project’s current status can be found on this dashboard.
